# Express_test
此專案為課程作業：Express基礎驗收

### install

1.打開terminal，Clone 此專案至本機電腦
```
git clone https://github.com/Keely1112/Express_test.git
```

2.開啟terminal
```
cd Express_test
```

3.安裝npm套件
```
npm install
```

4.安裝ejs模組
```
npm install ejs
```

5.啟動終端機
```
node app.js
```

6.出現以下內容表示伺服器已啟動
```
Express is listening on http://localhost:3000
```
